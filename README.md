# webflux-react-websocket-chat
Websocket chat app

## Stack
Backend:
- Spring WebFlux
- Redis - For chat room implementation

Frontend:
- React

## How to run
1. Install docker.
2. Clone repository.
3. Exploit docker compose with `docker compose up -d` at the proejct root directory.
4. Go to `http://localhost:3000` on browser and start chatting.
