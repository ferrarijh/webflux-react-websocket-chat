# webflux-react-websocket-chat
Websocket chat app sample with Spring WebFlux, Reactjs. (feat. docker compose)

## How to run
1. Install docker
2. Clone repository
3. `docker compose up -d`
4. Go to `http://localhost:3000` on browser and start chatting
